## Accuracy
Acc = TP + TN / TP + TN + FP + FN
## Precision
Precision = TP / TP+FP
## Recall
Recall = TP / TP+FN
## F1-score
F1 Score = 2*(Recall * Precision) / (Recall + Precision)

https://blog.exsilio.com/all/accuracy-precision-recall-f1-score-interpretation-of-performance-measures/
